# 📝 To-Do List App

A simple and beginner-friendly To-Do List web application built using **HTML**, **CSS**, and **JavaScript**.

This project allows users to:
- ➕ Add tasks
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks at once
- 🔍 Search tasks in real-time

---

## 🚀 Features

- Intuitive and clean UI
- Task list dynamically updates
- Delete icon for removing tasks
- Sticky header and input forms (stay at the top during scroll)
- Responsive behavior for long tasks (wraps to new line)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
